## 段落

Markdown 支持普通文本。为了在 Markdown 中创建一个新段落，你只需要在段落之间插入一个空行。

这是第二段。在 Markdown 中，段落可以很简单，不需要任何特殊的标记。

## 标题

Markdown 支持六级标题，使用 `#` 符号表示。

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

## 强调样式

*这是斜体文本*，使用单个星号或下划线包裹。

**这是加粗文本**，使用双星号或下划线包裹。

~~这是删除线文本~~，使用两个波浪线包裹。

## 列表

无序列表使用星号、加号或减号作为列表标记：

- 列表项一
- 列表项二
  - 子列表项一
  - 子列表项二

有序列表使用数字和点表示：

1. 第一项
2. 第二项
   1. 子项一
   2. 子项二

## 代码块

行内代码使用单个反引号包裹，例如：`<div></div>`。

代码块使用三个反引号包裹：

```javascript
function sayHello() {
    console.log("Hello, Markdown!");
}
```

## 引用

引用文本使用 `>` 符号：

> 这是一段引用文本。
>
> 这是引用的第二段。

## 链接和图片

链接使用 `[链接文本](URL)` 格式，例如：[OpenAI](https://openai.com)。

图片使用 `![替代文本](图片URL)` 格式，例如：

![Markdown Logo](https://images.unsplash.com/photo-1546984575-757f4f7c13cf?ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D&auto=format&fit=crop&w=2070&q=80)

## 表格

Markdown 表格使用竖线 `|` 和横线 `-` 来定义表头和单元格：

| 标题 1 | 标题 2 | 标题 3 |
|--------|-------|-------|
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

表格也可以对齐文本：

| 左对齐 | 居中  | 右对齐 |
|:------|:----:|------:|
| 文本  | 文本 | 文本  |
